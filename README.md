# tiktok-api-example

-   install [pipenv](https://pipenv.pypa.io/en/latest/installation.html) and
    [pyenv](https://github.com/pyenv/pyenv?tab=readme-ov-file#installation)
-   install dependencies with `pipenv install`
-   create `.env` file with `MS_TOKEN` env variable
-   run scraper `pipenv run python scrape.py`
